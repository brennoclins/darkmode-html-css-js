Light / Dark

O Modo escuro, em inglês dark mode, é uma forma diferente de ver as aplicações ou os sistemas operativos, mudando o aspecto de um tom claro para um tom escuro.

Isto não se trata de uma questão estética. Pois o objectivo é acima de tudo descansar os olhos dos utilizadores, especialmente para aqueles que usam mais esses dispositivos com ecrã (tela) de noite.

Com o modo escuro, diminui-se a difusão da luz azul emitida pelo ecrã (tela), que é frequentemente acusada de interromper os ciclos do sono.
